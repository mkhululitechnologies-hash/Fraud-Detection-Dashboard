# 💳 Fraud Detection Dashboard (Excel + Python)

## 📌 Project Overview

This project is an interactive fraud detection dashboard built using **Microsoft Excel and Python**. It analyzes financial transaction data to identify fraud patterns, high-risk behavior, and key business insights.

---

## ⚙️ Tools & Technologies

* Microsoft Excel (Pivot Tables, Charts, Slicers)
* Python (Pandas)
* Data Visualization
* Risk Scoring Model

---

## 🚀 Key Features

* 📊 Interactive dashboard with slicers and KPIs
* ⚠️ Fraud detection analysis (patterns, locations, transaction types)
* 🔍 High-risk transaction identification
* 🤖 Python-based risk scoring model
* 📈 Dynamic visual insights for decision-making

---

## 🤖 Machine Learning Component

A Python script (`risk_model.py`) was used to generate a **risk scoring model**, classifying transactions into:

* High Risk
* Medium Risk
* Low Risk

The model considers:

* International transactions
* Unusual transaction timing
* Previous fraud history
